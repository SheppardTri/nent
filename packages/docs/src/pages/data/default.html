Data System

Provide multiple ways to use data within your HTML.

How it works

The <n-data> component extends the functionality of other elements by introducing a data-provider system along with an expression-resolution system for using data in a simple, declarative way.

Once enabled data can be injected in HTML using tokens: 

{{source:key}}

Content Repeat Example

Assuming a list of URLs visited are stored within the browser's built-in storage, you can render that list in your HTML by enabling the <n-data> system and installing the <n-data-storage> provider.

The <n-content-repeat> element can then reference the list using the expression: 

{{storage:visits}}
