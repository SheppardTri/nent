Dynamic Routes

Data-driven, dynamic routing for HTML.

How it Works

This router supports segment and query-string variables and we provide a simple, declarative method for using that data in your application. By using special (Node.js Express-based) notation in the <n-view> element's URL attribute, the route data becomes available in our unified data expression system.

Routes can express dynamic parameters, to provide data-driven views You can even use that data in your HTML, again with no code.
